TsValuesSingleVariableTypea aggregates the list of values and associated metadata. It is the values element in the timeSereisResponse Attributes are optional, but use @count is encouraged. The atrributes @unitsAreConverted, @untsCode,@unitsAbbreviation, and @unitsType were originally included to allow for translation from orignal variable units. Thier use is not encouraged.
