Class SiteInfoType

source code

A sampling station is any place where data are collected. SiteInfoType is the Element that for the core information about a point sampling location. The core information includes SiteName, SiteCode(s), location, elevation, timeZone information and note(s). SiteInfoType is <siteInfo> in a <site> of a <sitesResponse>. It is derived from SourceType so that other geographic location descriptions can be utilized in the <sourceInfo> of the <timeSeriesResponse>
